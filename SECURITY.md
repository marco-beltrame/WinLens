# Security Policy

## Supported versions

The latest release on the [Releases](https://github.com/marco-beltrame/WinLens/releases) page
is the supported version.

## Reporting a vulnerability

If you discover a security issue, please **do not open a public issue**. Instead, report it
privately via GitHub's [private vulnerability reporting](https://github.com/marco-beltrame/WinLens/security/advisories/new)
or by contacting the maintainer directly.

Please include:

- A description of the issue and its impact.
- Steps to reproduce.
- Any relevant logs or screenshots.

You can expect an acknowledgement within a few days. Thanks for helping keep WinLens and its
users safe.

## Privacy note

WinLens sends the **recognized text** of the region you translate to an online translation
service (Google endpoint, with a MyMemory fallback). Screenshots are processed locally and
never uploaded. Offline translation is on the [roadmap](README.md#roadmap).
